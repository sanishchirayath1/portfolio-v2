# uses

The stuff I actually use. Not affiliate links, not aspirational.

## editor

- **Neovim** for everything text-first.
- **VS Code** when I want the LSP + debugger combo out of the box.

## terminal & shell

- **Fish** as my daily shell.
- **iTerm2** on macOS.
- **JetBrains Mono** everywhere a monospace shows up.

## languages I reach for

- **TypeScript**: the default for shipping.
- **Go**: when I want to think about the network without ceremony.
- **Python**: quick data work, scripting.

## day-to-day

- **Bruno**: API client (also where I work).
- **Raycast**: launcher, snippets, clipboard history.
- **Arc**: browser.
- **Linear**: issue tracking.
- **Obsidian**: notes, second brain.

## hardware

- MacBook Pro (M-series).
- External monitor + a mechanical keyboard I keep tuning.

## running / training

- **Garmin** for swim + cycle data.
